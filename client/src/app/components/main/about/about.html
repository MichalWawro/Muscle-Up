About:

MuscleUp! is an open-source web application designed to help you craft your perfect workout,

depending on which muscle group you want to focus on.

To use the app, simply select the muscle group you're targeting, and

it will suggest three exercises using different types of equipment -

from full-on gym equipment, through simple weights or a pull-up bar, to only your bodyweight.

We know how overwhelming it can be to start training with no prior experience, so head to the

training tab for workouts tailored to different skill levels - from a total beginner to a weight-lifting pro!
